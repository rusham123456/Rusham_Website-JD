Assignment 4

Microcontroller Programming

About this project

Arduino Uno project that makes a sound and lights an LED when a key is pressed. This project is a great way to learn about programming with an Arduino Uno and can be easily modified to incorporate additional features.

What I learned

I learned about what all things are on an Arduino and about different kinds of components one can connect and setup to perform various things. I also learned about the circuit and how to use a breadboard in order to correctly connect LEDs, buzzer and keypad membrane to Arduino Uno. Then I also learned to program the arduino to perform the required tasks and understood the logic behind the coding it.

Componentes needed

Arduino Uno, Breadboard, Buzzer, Keypad, 3 LEDs, jumper wires.

Steps

1.Connect the keypad to the breadboard, with the rows connected to pins 8, 7, 6, and 5 on the Arduino Uno, and the columns connected to pins 4, 3, 2, and 1 on the Arduino Uno, respectively.
2. Connect the 3 LEDs to the breadboard with a (-ve) end to ground. Then each indivisual (+ve) end to pins 10,11,12.

3. Connect the speaker to the Arduino on pin 13 and the other black wire to ground.
