Assignment 2

Basic Website Development

Introduction

While I have learnt basics of HTML and CSS in school. It was fun to revisit this and get a deeper insight on web development. The template was provided to us by Prof. Jay Dhariwal. I have analysed the HTML and CSS code and have learnt a lot about what went into the structure and aesthetics of the website.

Vison

My vison was to allow people to easily navigate and view my assignments and also learn about me. I changed the background photo of the home page, changed the colour scheme to more suit me.

Softwares Used

Visual Studio Code is a free source-code editor developed by Microsoft that provides a modern and customizable environment for coding in various programming languages. It includes features such as debugging, syntax highlighting, intelligent code completion, and support for extensions. When coding HTML and CSS for a basic website, Visual Studio Code helped by providing real-time feedback on any syntax errors, highlighting potential issues and offering helpful suggestions. The integrated terminal also allowed me to quickly test and preview my website as I made changes to the code. Overall, Visual Studio Code was a powerful and user-friendly tool for creating a website.

Process

An HTML/CSS template was provided to us, and it was simple to update because it was guided with comments. With the assistance of Internet, I attempted to update the about page and replace the background image in the provided template. The assignment page needed to be edited, images added, and a hyperlink created text.By modifying the CSS code, I started personalising the design. This involved updating the typeface to be more unified, the colour palette. I also changed the HTML to customise the content to meet my needs by adding more subheadings and paragraphs.

Learnings

Stronger understanding of layout, colour palettes, typography, and user experience as they relate to web design. Acquired increased familiarity with the HTML and CSS languages and a better comprehension of how they interact to generate a web page. During the modification phase, found creative solutions to deal with design and coding issues. Briefly, here is a few things I learned: - Syntax of HTML and CSS Tags like: - Div - A href - P - H1,2,3,4,5,6 One other interseting learning was how to use "inpect" by right-clicking a website and get information about the html and css code behind that website.

Problems Faced

1. Technical issues with website templates or the coding 2. HTML/CSS templates may have limitations that make customization of design elements difficult. 3. Learning how to edit and how to connect css and html 4. I had trouble coming up with a design that was both aesthetically pleasing and user-friendly. 5. Ensuring website is accessible to all users. 6. Keeping files organized.
